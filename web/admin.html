Admin Interface

All Actions on this Page are logged with your UserName

Install - Sets up the folder structure and Required DB Tables if not already Set

Backup - Archives the Database and associated Files

Reset- Erases all Stored Files and resets the Database to Default Values

Validate Logs- Validate the MD5 Sum of the Log Files

Email Logs- Parse Email Logs in to the Database

Web Logs- Parse Web Logs in to the Database

Delete Records To delete a single entry leave the End Blank

Start: End: - Delete the record and any associated Files
